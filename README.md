# Giggrid

## 🚀 Overview
Giggrid is a full-stack MERN application that allows users to post projects and place bids. It uses MongoDB Atlas for database management and provides a seamless project marketplace experience.

## 🛠 Tech Stack
- React.js (Frontend)
- Node.js (Backend)
- Express.js
- MongoDB Atlas

## ✨ Features
- User authentication
- Project posting
- Bidding system
- Full-stack integration

## 📁 Project Structure
Giggrid-Mongodb/
│
├── client/        # React frontend
├── server/        # Node/Express backend
├── .gitignore
├── README.md

## ⚙️ Installation

git clone https://github.com/dhibith-student/Giggrid-Mongodb.git  
cd Giggrid-Mongodb  

## ▶️ Run the Application

# Backend
cd server  
npm install  
npm start  

# Frontend
cd client  
npm install  
npm start  

## 🔐 Environment Variables

Create a `.env` file in the server folder and add:

MONGO_URI=your_mongodb_connection_string  
PORT=5000  

## 📌 Note
Make sure MongoDB Atlas is configured and your IP is whitelisted.
